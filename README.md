# Duolingo Domination Discord Bot

## About

This bot displays the leaderboard of our Duolingo Domination challenge on Discord.

## Usage

### Commands

- `/duo` - Displays the leaderboard

## Development

### Requirements

- [Node.js](https://nodejs.org/en/) (>= 18)
- [Pnpm](https://pnpm.io/) (>= 8)
